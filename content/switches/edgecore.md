The Edge-corE 5600-52X is a white-box switch built on the **Broadcom Trident+** (BCM56840) ASIC.

| Field | Value |
|-------|-------|
| Ports | 48 SFP+ and 4 QSFP |
| RX Queues | 8Q |
| Total Buffer | 9 MB |
| TX Buffer (max to 1 port) | ~5 MB (estimated) |

See the [Trident+ architecture page](/concepts/trident-plus/) for details on how the 9 MB shared buffer pool works.
