The **FS N8000-32Q** is a 1RU bare-metal data center switch built on the **Broadcom BCM56850 (Trident 2)** ASIC. It targets 40G spine and aggregation deployments.

| Field | Value |
|-------|-------|
| ASIC | Broadcom BCM56850 (Trident 2) |
| Ports | 32x 40G QSFP+ |
| Switching Capacity | 2.56 Tbps |
| Total Buffer | 12 MB |
| Architecture | Shared memory (on-chip SRAM) |
| TX Queues | 8 per port |

## Buffer Architecture

The N8000-32Q uses the same BCM56850 silicon found across many 40G spine platforms. The 12 MB buffer is divided between a small per-port dedicated region and a larger dynamic shared pool, consistent with all Trident 2 implementations. The dynamic pool is available to any port experiencing congestion.
