The Cisco Nexus 5548P uses **Virtual Output Queuing (VoQ)**.

| Field | Value |
|-------|-------|
| Ports | 32 fixed SFP+ and 16-port expansion module |
| RX Queues | 8 Unicast + 8 Multicast |
| TX Queues | 1 (uncertain) |
| Buffer per Port | 640 KB |
| RX Buffer | 640 KB |
| Architecture | VoQ |
