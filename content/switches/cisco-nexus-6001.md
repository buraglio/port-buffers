The Cisco Nexus 6001 uses **Virtual Output Queuing (VoQ)**.

| Field | Value |
|-------|-------|
| Ports | 48 fixed SFP+ and 4 QSFP |
| RX Queues | 8 Unicast + 8 Multicast |
| TX Queues | 1 (uncertain) |
| Total Buffer | 25 MB per 12-port 10G ASIC |
| RX Buffer | 14 MB shared per ASIC |
| Architecture | VoQ |
